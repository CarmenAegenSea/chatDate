# 数据库表结构 — chat_app

## 1. `users` — 用户表

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 用户 ID |
| `phone` | VARCHAR(20), UNIQUE | 手机号 |
| `nickname` | VARCHAR(32), NOT NULL | 昵称 |
| `avatar` | VARCHAR(500) | 头像 URL |
| `password` | VARCHAR(128) | BCrypt 加密密码 |
| `balance` | BIGINT | 余额 |
| `oauth_provider` | VARCHAR(20) | 第三方登录提供商 (wechat/qq) |
| `oauth_open_id` | VARCHAR(64) | 第三方 OpenID |
| `created_at` | DATETIME, NOT NULL | 注册时间 |
| `updated_at` | DATETIME | 更新时间 |
| UNIQUE | (oauth_provider, oauth_open_id) | 第三方登录唯一约束 |

---

## 2. `posts` — 帖子表

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 帖子 ID |
| `title` | VARCHAR(200) | 标题 |
| `content` | TEXT | 内容 |
| `user_id` | BIGINT (FK → users.id) | 作者 ID |
| `mood` | VARCHAR(50) | 心情标签 |
| `like` | BIGINT | 点赞数 |
| `bookmark_count` | INT | 收藏数 |
| `comment_count` | INT | 评论数 |
| `created_at` | DATETIME | 创建时间 |
| `updated_at` | DATETIME | 更新时间 |

---

## 3. `post_likes` — 帖子点赞关联表

| 字段 | 类型 | 含义 |
|------|------|------|
| `post_id` | BIGINT (FK → posts.id) | 帖子 ID |
| `user_id` | BIGINT (FK → users.id) | 用户 ID |

---

## 4. `post_bookmarks` — 帖子收藏关联表

| 字段 | 类型 | 含义 |
|------|------|------|
| `post_id` | BIGINT (FK → posts.id) | 帖子 ID |
| `user_id` | BIGINT (FK → users.id) | 用户 ID |

---

## 5. `comments` — 评论表

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 评论 ID |
| `post_id` | BIGINT (FK → posts.id) | 所属帖子 ID |
| `user_id` | BIGINT (FK → users.id) | 评论作者 ID |
| `content` | TEXT | 评论内容 |
| `like_count` | INT | 点赞数 |
| `created_at` | DATETIME | 创建时间 |

---

## 6. `comment_likes` — 评论点赞关联表

| 字段 | 类型 | 含义 |
|------|------|------|
| `comment_id` | BIGINT (FK → comments.id) | 评论 ID |
| `user_id` | BIGINT (FK → users.id) | 用户 ID |

---

## 7. `games` — 游戏表（联机大厅用）

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 游戏 ID |
| `name` | VARCHAR(100) | 游戏名称 |
| `icon` | VARCHAR(255) | 图标 |
| `current_players` | INT | 当前频道人数 |
| `created_at` | DATETIME | 创建时间 |

预置 6 款游戏：王者荣耀、和平精英、原神、英雄联盟、绝地求生、永劫无间

---

## 8. `game_options` — 游戏选项表（备选游戏池）

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 主键 |
| `code` | VARCHAR(20), UNIQUE | 游戏代码 (如 LOL, WZRY) |
| `name` | VARCHAR(100), NOT NULL | 游戏名称 |
| `category` | VARCHAR(50) | 分类 (MOBA/FPS/RPG 等) |
| `sort_order` | INT | 排序序号 |

预置 21 款游戏，涵盖 MOBA/FPS/Action/RPG/Strategy/Casual 六大分类

---

## 9. `game_players` — 用户游戏资料表

| 字段 | 类型 | 含义 |
|------|------|------|
| `user_id` | BIGINT (PK+FK → users.id) | 用户 ID（与 User 共享主键） |
| `is_companion` | BIT, NOT NULL | 是否愿意成为陪玩 |
| `bio` | VARCHAR(200) | 陪玩简介 |

---

## 10. `game_player_favorites` — 用户喜欢的游戏关联表

| 字段 | 类型 | 含义 |
|------|------|------|
| `user_id` | BIGINT (FK → game_players.user_id) | 用户 ID |
| `game_id` | BIGINT (FK → games.id) | 游戏 ID |

---

## 11. `players` — 陪玩账号表

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 陪玩账号 ID |
| `user_id` | BIGINT (FK → users.id), UNIQUE | 关联的用户 ID |
| `bio` | VARCHAR(500) | 个人简介 |
| `hourly_rate` | INT | 每小时陪玩价格（元） |
| `status` | VARCHAR(20) | 状态: ONLINE / OFFLINE / BUSY |
| `total_duration` | BIGINT | 总计陪玩时长（分钟） |
| `created_at` | DATETIME, NOT NULL | 创建时间 |
| `updated_at` | DATETIME | 更新时间 |

---

## 12. `player_games` — 陪玩-游戏关联表

| 字段 | 类型 | 含义 |
|------|------|------|
| `id` | BIGINT (PK) | 主键 |
| `player_id` | BIGINT (FK → players.id) | 陪玩账号 ID |
| `game_code` | VARCHAR(20), NOT NULL | 游戏代码（关联 game_options.code） |
| `skill_level` | VARCHAR(20) | 技术水平: 入门/熟悉/精通/大神 |
| UNIQUE | (player_id, game_code) | 一个陪玩一个游戏只能有一条记录 |

---

## 两大陪玩系统说明

该项目有两套平行的陪玩机制：

- **旧系统** (`game_players` + `games`)：通过 `GamePlayer.isCompanion` 标记，喜欢的游戏关联 `games` 表，走 `GameService`/`GameController`，用于联机大厅和游戏详情
- **新系统** (`players` + `player_games`)：独立的陪玩账号，关联 `game_options.code`，走 `PlayerService`/`PlayerController`，时薪/状态/总时长更完整
